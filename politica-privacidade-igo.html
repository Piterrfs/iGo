Política de Privacidade - iGo

Última atualização: 24 de novembro de 2025

1. Informações que Coletamos

O aplicativo iGo coleta informações fornecidas pelo usuário, incluindo:

  • Números de telefone para envio de mensagens via WhatsApp Business API
  • Endereços de email para envio de notificações via Gmail
  • Informações sobre transmissões ao vivo (links, tipo de transmissão)
  • Mensagens personalizadas fornecidas pelo usuário

2. Como Usamos as Informações

As informações coletadas são usadas exclusivamente para:

  • Envio de notificações sobre transmissões ao vivo iniciadas
  • Comunicação com usuários via WhatsApp Business API e Gmail
  • Melhoria dos serviços oferecidos
  • Processamento de requisições de envio de mensagens

3. Compartilhamento de Informações

Não compartilhamos informações pessoais com terceiros, exceto quando necessário para fornecer os serviços:

  • WhatsApp Business API (Meta): Para envio de mensagens via WhatsApp
  • Gmail (Google): Para envio de emails
  • n8n: Para processamento e automação de envio de mensagens

Todos os serviços utilizados seguem suas próprias políticas de privacidade e termos de serviço.

4. Segurança

Implementamos medidas de segurança para proteger suas informações pessoais:

  • Dados são transmitidos usando conexões seguras (HTTPS)
  • Tokens de acesso são armazenados de forma segura
  • Acesso limitado apenas a pessoal autorizado

5. Retenção de Dados

Os dados são processados em tempo real para envio de mensagens e não são armazenados permanentemente após o processamento, exceto quando necessário para fins de auditoria ou conformidade legal.

6. Seus Direitos

Você tem o direito de:

  • Acessar suas informações pessoais
  • Corrigir informações incorretas
  • Solicitar exclusão de suas informações
  • Revogar consentimento a qualquer momento

7. Cookies e Tecnologias Similares

Este serviço utiliza tecnologias de terceiros (WhatsApp Business API, Gmail, n8n) que podem usar cookies ou tecnologias similares. Consulte as políticas de privacidade desses serviços para mais informações.

8. Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. A data da última atualização está indicada no topo desta página. Recomendamos que você revise esta política periodicamente.

9. Menores de Idade

Este serviço não é destinado a menores de 18 anos. Não coletamos intencionalmente informações de menores de idade.
